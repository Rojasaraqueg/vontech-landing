Tecnologia de siguiente generacion

Vontech desarrolla sistemas HMI y plataformas IoT para distribuidores y fabricantes de plantas de tratamiento de agua, sistemas de piscinas y equipos industriales en general. Integramos hardware y software para ofrecer control total: desde la operacion en planta hasta la gestion de clientes, tecnicos y servicios.

No fabricamos maquinas. Construimos la tecnologia de control y monitoreo que las hace inteligentes.

Machine Editor
Machine Editor

Disena tus procesos visualmente

El Machine Editor permite disenar la secuencia completa de control de cualquier maquina. Arrastra y conecta componentes industriales para crear flujos de proceso personalizados con logica avanzada.

  • Setpoints Configurables — Cada sensor define umbrales que disparan acciones automaticas
  • Nodos y Dependencias — Logica entre sensores para secuencias complejas
  • Secuencias Multi-Paso — Backwash, CIP, llenado con pasos temporizados y condiciones
